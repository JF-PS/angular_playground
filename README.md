###### create component :

`Create Component :`
`npx nx g @nrwl/angular:component techno-list --skip-import --project=playground`

###### create service :

`Create Service :`
`npx nx g service techno --project=playground`

##### add angular material :

`npx nx g @angular/material:ng-add --project=playground`

npm run ng -- deploy
